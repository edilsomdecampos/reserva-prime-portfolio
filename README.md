# 🏛️ Reserva Prime | High-End Real Estate Interface

Este projeto é uma vitrine de **UI/UX de Luxo** desenvolvida para o mercado imobiliário de alto padrão. O foco principal foi a criação de uma interface que traduzisse exclusividade e sofisticação através de escolhas tipográficas e ritmos visuais precisos.

## 🚀 Diferenciais Técnicos
- **Design System "Quiet Luxury":** Implementação de variáveis CSS para controle total de paleta bronze/ouro (#b08d57) e tipografia clássica (*Cormorant Garamond*).
- **Arquitetura Minimalista:** Código limpo e semântico, priorizando a performance e a legibilidade.
- **Deploy de Alta Performance:** Integrado via Vercel com automação contínua.

## 🛠️ Tech Stack
- **Linguagem:** HTML5 Semântico
- **Estilo:** CSS3 (Modern Flexbox & Grid)
- **Hospedagem:** Vercel (Production-Ready)
